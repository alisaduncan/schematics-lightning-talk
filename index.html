Angular Schematics

What is it?

A schematic is a template-based code generator that supports complex logic. It is a set of instructions for transforming a software project by generating or modifying code.

Why do we want it?

You've already used it. It allows us to make code changes very quickly.

An example

ng generate component my-component

What does this schematic do?

  • Creates an Angular component
  • Imports the component into AppModule

What else can schematics do?

  • Create a brand new Angular app using ng new
  • Add library dependencies using ng add
  • Update your app using ng update
  • Run schematics published by other groups, such as Angular Components

So much power!

Let's look a little closer at Material

Add Angular Material to your app

ng add @angular/material
  • Update your project dependencies
  • Add some fonts, icons, and styles
  • Enable a theme and built-in animations

Adding a dashboard

ng generate @angular/material:dashboard my-dashboard-component

What else is available?

  • Adding an address form
  • Adding navigation
  • Adding a table
  • Drag and drop support for creating an interactive to-do list
